二叉树前中后序遍历的三种实现

发表于 2020-09-07 阅读次数：
本文字数： 5.6k 阅读时长 ≈ 5 分钟

二叉树遍历

  • 前序遍历：先访问根节点，再前序遍历左子树，再前序遍历右子树。
  • 中序遍历：先中序遍历左子树，再访问根节点，再中序遍历右子树。
  • 后序遍历：先后序遍历左子树，再后序遍历右子树，再访问根节点。

前序遍历的三种实现

  1. 递归实现中，只需修改 dfs 的遍历顺序即可。

2.非递归实现中，采用栈的形式时：时间复杂度：O(N)；空间复杂度是 O(N)。

3.莫里斯遍历：

算法流程：

  1. 初始化 TreeNode * node = root,* pre=NULL;
  2. 判断 node.left==NULL;
    • 若等于：将node 添加到输出；新的 node = node.right;
    • 若不等于：pre继续往左一次，然后持续往右直到遇到NULL(跳转到第3步) 或 当前node（跳转到第4步）；
  3. 遇到了NULL：将当前 node 添加到输出；将叶子结点pre.right 赋值为当前node；新的 node = node.left;
  4. 遇到了当前node，删除遍历到叶子结点 pre.right；新的node = node.right；

时间复杂度：每个前驱（pre）恰好访问两次，因此复杂度是 O(N)，其中 N 是顶点的个数，也就是树的大小。

空间复杂度：我们在计算中不需要额外空间，但是输出需要包含 N 个元素，因此空间复杂度为 O(N)。

中序遍历的三种实现

1.递归实现，修改dfs的遍历顺序

2.非递归实现，借助栈 ；时间复杂度：O(n)。空间复杂度：O(n)。

3.莫里斯中序遍历

算法流程：

  1. 初始化 TreeNode * node = root,* pre=NULL;

  2. 判断 node.left==NULL;

    • 若等于：将node 添加到输出；新的 node = node.right;
    • 若不等于：pre继续往左一次，然后持续往右直到遇到NULL(跳转到第3步) 或 当前node（跳转到第4步）；

  3. 遇到了NULL：将叶子结点pre.right 赋值为当前node；新的 node = node.left;

  4. 遇到了当前node：将当前 node 添加到输出；删除遍历到叶子结点 pre.right；新的node = node.right；

后序遍历的三种实现

1.递归实现，修改dfs的遍历顺序

2.非递归实现，借助栈，但有些取巧，只是能返回遍历的结果，并不是严格意义上树拓扑结构的遍历。虽然结果是正确，但是如果需要按照后续遍历的顺序对树节点进行访问（或操作），此法就无法满足。

  • 时间复杂度：O(n)。空间复杂度：O(n)。

  • 类似于前序遍历，只不过输出序列要在首部插入值（起到逆序的效果）

3.非递归实现，严格按照后序遍历的定义，需要一个额外的 set 来记忆已经访问过的结点。
