函数模板与类模板

发表于 2020-08-18 阅读次数：

函数模板

  • 在理解模板之前，首先引进 泛型程序设计（Generic Programming）的概念：
    • 算法实现时不指定具体要操作的数据的类型，在实际使用算法时再给出具体的数据类型；
    • 泛型：算法实现一遍，可适用于多种数据结构；
    • 优势: 减少重复代码的编写；
    • C++ 中实现泛型程序设计的工具，就是模板；
    • 大量编写模板, 使用基于模板的程序设计可以分为两类：函数模板 和 类模板。

类的继承

  • 在定义一个新的类B时，如果该类与某个已有的类A相似(指的是B拥有A的全部特点)， 那么就可以把A作为一个基类，而把B作为基类的一个**派生类(也称子类)**。
  • 派生类是通过对基类进行修改和扩充得到的。在派生类中，可以扩充新的成员变量和成员函数。

基本概念说明

  • C++预定义了运算符表示对数据的运算：+, -, *, /, %, ^, &, ~, !, |, =, <<, >>, != ；
